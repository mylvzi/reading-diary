# 一.http消息的产生
浏览器本质上是一个**多功能的客户端软件**,可以作为网络服务客户端,邮件服务客户端,文件服务客户端
通过不同的url来行使不同的功能,比如**http**就代表浏览器是一个网络客户端, ftp代表浏览器是一个文件服务客户端

不能把浏览器当做一个简单的访问网址的软件

浏览器本身会生成http请求报文(请求头 + 首行 + 空行 + 主体),但是浏览器本身不具有**网络接口**功能,需要委托操作系统将http请求发送给相应的服务器

# 二.向DNS服务器发送消息获取IP地址
多个计算机如何连接起来 : 
**集线器**将多个单独 的计算机连接起来,组成一个子网
路由器负责将多个由集线器组织的子网整合在一起,形成一个局域网络

IP地址本质上是一个32bit位(4组, 圆点隔开)的数字
IP = 网络号 + 主机号, 通过子网掩码来进行区分, 255.255.255.0

如何根据域名获取ip地址呢?通过socket库来实现, socket库是负责网络传输的多个组件的集合, 通过**gethostbyname**, 获取到ip地址,并将ip地址存入到浏览器对应的内存地址中, 后续访问时, 浏览器会直接拿着存在内存地址中的ip地址进行访问
