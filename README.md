Reading Notes — 刘双奇的阅读记录

有时候，我会感觉日子像一条缓慢却毫不留情的河流，把人悄无声息地往下游推。工作、休息、刷手机——循环往复，像是在一条看不见的传送带上。
为了不让自己完全被运送出去，我决定在这条河的某个地方钉下一颗小钉子。
这个仓库，大概就是那颗钉子。

我叫 刘双奇。
本科就读于 吉林大学 · 地理信息科学专业。
毕业后，我去了美团做 QA，日常工作是测试、验证、复现、沟通，偶尔也会陷入一种机械化的透明感。
为了对抗它，我需要某种能让我感到“我还在这里”的东西——于是有了这个仓库。

🧭 这个仓库的意义

我并不是为了写出什么宏大、深刻、震撼谁的文字。
只是觉得，在日复一日的生活里，能把读过的书里某个打动我的句子、某个让人心里一紧的想法、某个我自己也说不清但隐隐约约有所触动的情绪，安安静静地放在这里，是件不错的事。

某种意义上，这也是一种自救。
一种让自己从虚无感里捡回一点碎光的动作。
如果能在写下它们的过程中找到一点点意义，那就再好不过了。

📚 内容说明

这里会放置我阅读时的：

某些片段的感悟

若隐若现的想法

没有答案的问题

对自己说的一些话

可能后来再读会觉得“这写的什么”的句子

它们不一定系统，也不一定正确；
它们只是属于我当时当刻的真实。

🌤 关于风格

你不会在这里看到严谨的学术论证，
也不会看到一板一眼的摘要式总结。
更像是在深夜的房间里，一个人喝着温水，听着不知道名字的歌，
随手写下的几段话。

如果读书是一种和世界对话的方式，那这里就是我为了不让这段对话断掉，而留下的文字痕迹。

🛠 技术上不复杂，但精神上重要

这个仓库没有框架、没有前端、没有服务端。
它只由文字构成，非常轻。
但它的意义，对我来说并不轻。

它提醒我：
即使生活像一张模糊的地图，我也可以自己在上面画线。

🌱 最后的话

如果你偶然来到这里，看到某些句子和你产生了轻微的共振，那大概就是这件事最好的结果了。
如果没有，也没关系 —— 它本来就是写给我自己的。

欢迎来到我的小小阅读角落。
—— 刘双奇
